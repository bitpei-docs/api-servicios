Catalogo Last updated: 2019-06-01

Metodos para consultar el catalago de transacciones y subcatalogos que son utilizados en los productos disponibles.

Productos(string)

Proporciona la lista de productos asignados al comisionista.

Parametros()

Proporciona el catalogo de parametros que son solicitados en productos financieros.

Cajero

Metodos para el registro/bajas de cajeros.

Registrar(DatosPersonales)

Registra un nuevo cajero, al completar el registro se generara el terminal_id que es utilizado en la ruta /api/transaccion/aplicar. El numero de terminal_id sera retornado en la propiedad mensaje.

Productos()

Proporciona la lista de cajeros activos.

Servicio

Metodos para consultar el estado del servicio y el saldo disponible del comisionista.

Saldo()

Este método proporciona información sobre el saldo disponible del comisionista, así como el monto total de ingresos y egresos transaccionados durante el día de operación. La línea de autorización disponible para el comisionista dependerá de la configuración asignada..

Transaccion

Metodos para la aplicacion de transacciones y consulta de estatus de la operacion.

ConfirmarTransaccion(string)

Comprueba el estado de una operacion, este endpoint puede ser consumido para validar el estatus de una operacion..

Metodos para la consulta de datos.
